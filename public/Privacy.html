Política de Privacidade

Última atualização: 14 de maio de 2026

1. Informações que Coletamos

Para fornecer nossos serviços, coletamos:

  • Dados de cadastro: nome, email e senha (armazenada com hash bcrypt)
  • Dados do seu negócio: informações de clientes, MRR, contratos que você inserir no sistema
  • Dados de uso: logs de acesso, ações realizadas no dashboard

2. Como Usamos seus Dados

Seus dados são usados exclusivamente para:

  • Fornecer as funcionalidades do NeuralOps (análise de churn, upsell, projeções financeiras)
  • Melhorar a qualidade dos agentes de IA
  • Comunicação sobre seu uso da plataforma

3. Compartilhamento de Dados

Não vendemos nem compartilhamos seus dados com terceiros para fins comerciais. Seus dados podem ser processados por:

  • Anthropic: para processar mensagens no chat de IA (conforme a política de privacidade deles)
  • Railway: infraestrutura onde o sistema é hospedado

4. Segurança

Implementamos as seguintes medidas de segurança:

  • Senhas armazenadas com bcrypt (hash seguro)
  • Autenticação via JWT com expiração
  • HTTPS em todas as conexões
  • Rate limiting para prevenir ataques

5. Seus Direitos

Você tem o direito de:

  • Acessar todos os dados que armazenamos sobre você
  • Solicitar a exclusão da sua conta e todos os dados associados
  • Exportar seus dados

Para exercer esses direitos, entre em contato através da página de contato.

6. Retenção de Dados

Mantemos seus dados enquanto sua conta estiver ativa. Após o cancelamento, os dados são excluídos em até 30 dias.
