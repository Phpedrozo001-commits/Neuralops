💳 Conectar Stripe

Sincronize automaticamente seus clientes, MRR e dados de assinatura do Stripe.

  • 1Acesse dashboard.stripe.com → Developers → API Keys
  • 2Copie a Secret key (começa com sk_live_ ou sk_test_)
  • 3Cole abaixo e clique em Salvar
  • 4Para webhooks, adicione a URL:
    https://neuralops-sage.vercel.app/api/webhooks/stripe

💬 Conectar Slack

Receba notificações no Slack quando os agentes criarem novas aprovações.

  • 1Acesse api.slack.com/apps → Create App
  • 2Ative Incoming Webhooks
  • 3Clique em Add New Webhook to Workspace e escolha o canal
  • 4Copie a URL do webhook e cole abaixo

💚 Conectar WhatsApp (Z-API)

Envie mensagens de retenção e upsell via WhatsApp. Requer conta Z-API (zapi.io).

  • 1Crie uma conta em z-api.io
  • 2Crie uma instância e conecte seu WhatsApp
  • 3Copie o Instance ID e Token e cole abaixo

⚡ Conectar via Zapier / Webhook

Envie dados de qualquer ferramenta para o NeuralOps via webhook. Funciona com Zapier, Make, n8n e qualquer sistema com webhooks.

https://neuralops-sage.vercel.app/api/webhooks/import

Envie um POST com JSON no formato:

{"name":"João Silva","email":"joao@empresa.com","mrr":500,"engagement_score":75}

No Zapier: Action → Webhooks by Zapier → POST → cole a URL acima.

🟠 Importar do HubSpot

Importe seus contatos do HubSpot para o NeuralOps via CSV. Sincronização automática em breve.

  • 1No HubSpot: Contatos → Exportar → Todos os contatos → CSV
  • 2Certifique-se de incluir as colunas: Nome, Email, MRR/Receita
  • 3Importe o CSV na nossa página de importação
IR PARA IMPORTAR CSV →

🔵 Importar do Pipedrive

Importe seus deals e contatos do Pipedrive para o NeuralOps.

  • 1No Pipedrive: (...) → Exportar dados → Deals ou Pessoas → CSV
  • 2Certifique-se de incluir: Nome, Email, Valor do Deal
  • 3Importe o CSV na página de importação
IR PARA IMPORTAR CSV →
