Integração via Webhook
Conecte Zapier, Make, n8n ou qualquer ferramenta de automação
Seu Endpoint Webhook
https://neuralops-sage.vercel.app/api/webhooks/import
Método: POST · Header: Authorization: Bearer {seu_token}
Formato do Payload (JSON)
[
  {
    "name": "TechFlow Ltda",
    "email": "contato@techflow.com",
    "mrr": 2800,
    "engagement_score": 65
  },
  {
    "name": "DataVision SA",
    "email": "admin@datavision.com",
    "mrr": 5500,
    "engagement_score": 72
  }
]
Zapier — Como configurar
1. Crie um novo Zap
2. Trigger: qualquer evento (novo CRM, planilha, etc.)
3. Action: Webhooks by Zapier
4. Method: POST · URL: acima
5. Headers: Authorization: Bearer {token}
6. Body: JSON com name, email, mrr
Make / n8n — Como configurar
1. Adicione módulo HTTP Request
2. URL: endpoint acima
3. Method: POST
4. Headers: Authorization: Bearer {token}
5. Body: array JSON de clientes
6. Teste e ative o cenário

API REST — Documentação
POST /api/customers — Adicionar 1 cliente
curl -X POST https://neuralops-sage.vercel.app/api/customers \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{"name":"Empresa X","email":"x@emp.com","mrr":2000,"engagement_score":70}'
POST /api/webhooks/import — Importar em lote
curl -X POST https://neuralops-sage.vercel.app/api/webhooks/import \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '[{"name":"A","email":"a@a.com","mrr":1000},{"name":"B","email":"b@b.com","mrr":2000}]'
GET /api/customers — Listar todos os clientes
curl https://neuralops-sage.vercel.app/api/customers \
  -H "Authorization: Bearer {token}"
Seu Token de API
Seu token JWT de sessão. Use no header Authorization: Bearer {token}
